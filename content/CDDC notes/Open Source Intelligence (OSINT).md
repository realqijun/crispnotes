OSINT Processes:
 1. Source Identification
	 1. Where can you find the information? (i.e. Google, Bing, Baidu)
 2. Data Harvesting
	 1. Get the relevant data from identified sources
 3. Data Processing and Integration
	 1. Process the acquired data and get the meaningful information
 4. Data Analysis
	 1. Join the data acquired from multiple sources
 5. Results Delivery
	 1. Create final report
