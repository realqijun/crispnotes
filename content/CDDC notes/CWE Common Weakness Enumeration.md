 - Weaknesses: each class contains specific weaknesses that describe a particular kind of vulnerability
### Hardware CWE
1. Growing importance: weaknesses in sectors such as IoT, automotive, and critical infrastructure
2. Patching or updating hardware components is more difficult compared to software
3. There is a need for standard classification
