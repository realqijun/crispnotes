Surface Web typically involves searching via Google or Bing. Beneath that is the Deep Web, which may occasionally be indexed by the Google. However, it requires a registered account and authentication.

### Deep Web
Can only be accessed through direct connection or by obtaining information from someone with access

### Dark Web
Even deeper and requires specific software to access it

Dark Web applications:
 - Tor
	 - Onion routing: Multihop routing with host key encryption
 - I2P (Invisible Internet Project)
 - Freenet
 - Zeronet
	 - Uses blockchain with URLs as bitcoin addresses

Deep Web Websites:
 - Nulled forum
 
Dark Web Websites:
 - Hidden Wiki
 - Dark Web MediaWiki
